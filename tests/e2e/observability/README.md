# Observability E2E Smoke Scaffolding

This directory contains the smoke-verification scaffold for the observability plane.

Current scope:

- validate the checked-in smoke matrix against the observability contracts,
- verify scraping coverage for the required subsystem roster,
- verify dashboard scope and dimension coverage,
- and verify health-state vocabulary, freshness, and scope isolation.

Deferred to later tasks:

- live environment probing,
- browser automation,
- screenshots or visual assertions,
- and runtime operator workflows that consume the smoke baseline.
